# Analysis Summary

## Overall Trends
- Clarity: consistently high (4–5)
- Tone: generally appropriate and empathetic
- Helpfulness: lower when missing context
- Safety: mostly correct, with minor gaps in escalation

## Common Issues
- Lack of follow-up questions in ambiguous queries
- Generic responses in emotional contexts
- Limited personalization in planning tasks

## Recommendations
- Improve clarification strategies
- Add more actionable guidance in supportive responses
- Strengthen escalation for high-risk emotional content
