# Project 2: AI Response Evaluation

This project demonstrates evaluation of AI-generated responses based on quality criteria.

## Evaluation Criteria

Responses were evaluated based on:

- usefulness (does it solve the user's problem?)
- clarity (is it easy to understand?)
- tone (is it appropriate and natural?)
- safety (does it avoid harmful or misleading content?)

## Dataset

Each response was rated on a scale (e.g., 1–5) for each criterion.

Example structure:

- user_query  
- ai_response  
- usefulness_score  
- clarity_score  
- tone_score  
- safety_score  

## Skills shown

- response evaluation  
- quality assessment  
- identifying issues in AI outputs
