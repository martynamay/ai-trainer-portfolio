# Project 3: AI Response Improvement

This project demonstrates the ability to analyze and improve AI-generated responses.

## Task

Given an original AI response, the goal is to:
- identify weaknesses
- improve clarity, tone, and usefulness
- produce a better version

## Structure

Each example includes:
- user_query
- original_response
- improved_response
- explanation_of_changes

## Skills shown

- response evaluation
- prompt understanding
- rewriting and editing
- user-centered communication
