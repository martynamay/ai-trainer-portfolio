# AI Trainer Portfolio

This portfolio contains projects demonstrating skills in AI training, annotation, and evaluation.

## Project 1: Intent & Entity Annotation Dataset

This project includes a dataset of user utterances annotated with:

- intent (user goal)
- entity_type (category)
- entity_value (exact phrase from input)

### Structure

- dataset/intent_entity_dataset.csv → annotated data
- documentation/annotation_guidelines.md → annotation rules
- analysis/annotation_notes.md → insights and observations

### Key Decisions

- Used snake_case for intents and entity types
- Used natural language for entity values
- Applied consistent labeling across domains
